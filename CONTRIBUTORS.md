# Contributors

Sermon Works is a caretaker restoration of the dormant [Sermon Manager](https://github.com/WP-for-Church/Sermon-Manager) plugin by WP-for-Church. This document credits everyone whose work lives on in Sermon Works, whether through direct code contributions, issue reports that led to fixes, or influence on the plugin's original design.

## Principle

Every code contribution from an upstream Pull Request is preserved in the git commit history with the original author's identity intact. This file is a human-readable companion to that record, not a replacement for it — the ledger is the repository itself.

## Original authors and maintainers (pre-fork)

The Sermon Manager plugin was created and maintained by contributors to the [WP-for-Church](https://github.com/WP-for-Church) organisation before falling dormant in 2019. All of their work is the foundation of Sermon Works. Full historical git log preserved in this repository's commit history.
